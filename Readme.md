## About

Through this course, I learned core JavaScript concepts, DOM manipulation, events, OOP, and localStorage.  
Each folder in this repo represents a major topic I covered while practicing and building projects.

## Projects

- **Guess the Number** – A number guessing game built with loops and conditionals.  
- **NoteNest (Final Project)** – A notes app built using HTML, CSS, and JS.  
  It allows creating, viewing, and deleting notes stored in localStorage.
